void API.Discord.GetCurrentUserAvatar ( Integer  size,
Function  callback 
)

Gets the current user's avatar. This is an asynchronous operation. The callback function must have one parameter, and it receives an Image if the operation succeeded, or.

null if the operation failed. The integration must have been initialized with API.Discord.Init before calling this.

Parameters
sizeThe size of the avatar to fetch. Must be one of: 16, 32, 64, 128, 256, 512, 1024
callbackThe callback to execute.

Parameters
